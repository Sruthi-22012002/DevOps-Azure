## Run the application
### Move to frontend directory
```bash
~/ems-ops-phase-0/react-hooks-frontend$
```
### Install npm to start the frontend
```bash
npm install
```
### Run the application
```bash
npm start
```
### Run the application in browser
> [http://localhost:3000](http://localhost:3000)
